# Pizza Nova — сайт для замовлення піци

Готовий сайт з сучасним дизайном, меню, фотографіями позицій, корзиною та записом замовлень у SQLite-базу.

## Файли

- `index.html` — головна сторінка сайту
- `style.css` — усі стилі
- `script.js` — меню, корзина, оформлення замовлення
- `server.js` — backend, який приймає замовлення та записує їх у базу
- `admin.html` — проста сторінка перегляду замовлень
- `admin.js` — завантаження замовлень з бази
- `database.sqlite` — створиться автоматично після запуску сервера
