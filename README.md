# Pizza Nova — сайт для замовлення піци

Готовий сайт з сучасним дизайном, меню, фотографіями позицій, корзиною та записом замовлень у SQLite-базу.

## Файли

- `index.html` — головна сторінка сайту
- `style.css` — усі стилі
- `script.js` — меню, корзина, оформлення замовлення
- `server.js` — backend, який приймає замовлення та записує їх у базу
- `admin.html` — проста сторінка перегляду замовлень
- `admin.js` — завантаження замовлень з бази
- `database.sqlite` — створиться автоматично після запуску сервера

## Як запустити

1. Встанови Node.js.
2. Відкрий папку проєкту в терміналі.
3. Виконай:

```bash
npm install
npm start
```

4. Відкрий сайт:

```text
http://localhost:3000
```

5. Сторінка замовлень:

```text
http://localhost:3000/admin.html
```

## Як змінити меню

У файлі `script.js` знайди масив:

```js
const menuItems = [...]
```

Там можна змінювати:
- назву позиції
- категорію
- ціну
- розмір
- опис
- фото

## Важливо

Якщо просто відкрити `index.html` подвійним кліком, корзина буде працювати, але запис у базу даних не спрацює. Для бази треба запускати сайт через backend:

```bash
npm start
```
